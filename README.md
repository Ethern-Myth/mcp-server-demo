# MCP Server Demo

A demonstration of creating a Model Context Protocol (MCP) server that can be easily containerized with Docker.

## 🚀 Overview

This project showcases how simple it is to build an MCP server and integrate it with Docker-based MCP architectures. It serves as a practical example for developers looking to understand MCP server implementation.
